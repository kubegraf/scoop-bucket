# KubeGraf Scoop Bucket

Scoop bucket for installing KubeGraf on Windows.

## Installation

```powershell
scoop bucket add kubegraf https://github.com/kubegraf/scoop-bucket
scoop install kubegraf
```

## About KubeGraf

Intelligent Kubernetes Control Center - A powerful CLI and web interface for managing Kubernetes clusters.

Visit [kubegraf.io](https://kubegraf.io) for more information.
